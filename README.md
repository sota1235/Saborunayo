Saboruna Yo
====

開発サボってるとYoが来る監視用Yoアカウント

### Description

GitHubのContributionが0だと夜9時にYoしてくる。

Yoが来たらpushなり開発なりすればよいとおもう

### Usage

GitHubアカウント名と同じYoアカウントで`SABORUNAYO`にYoを１回送るだけ
